# Features

## Customer

- Login/Signup(using firebase)
- Place orders
- View Order history
- Search Restaurants using name or location

## Restaurant

- Login/Signup(using firebase)
- Add/Update/Delete menu item(s)
- View order history
- Accept orders

## Admin

- Login/Signup(using firebase)
- Approve restaurants
- View reports
